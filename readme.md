# Quick start 

## INSTALL

### Requirements

- Docker & docker-compose


### Run the app in dev env 

- Open a batch terminal
- RUN 'cd /your directory'
- RUN 'git clone https://github.com/gjosso/map'
- RUN 'cd /map'
- RUN 'docker-compose up'
- Go to localhost:8080
